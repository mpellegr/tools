# Font Replacer

ScriptUI font replacement tools for scanning fonts used in AE text layers and replacing them with selected or hardcoded target fonts.

## Contents

- `FontReplacer.jsx`, `FontReplacer_v02.jsx`, `FontReplacerMercury.jsx`: general font replacement panels.
- `FontReplaceTo_ApercuMovistarPlus.jsx`, `FontReplaceTo_BebasNeuePro.jsx`, `FontReplaceTo_VarienRegular.jsx`: target-font-specific replacement scripts.

These scripts traverse project comps, read `Source Text` / `TextDocument`, and update font properties.
